# Download

## Installation

1. Install Minecraft 1.21.1
2. Install Fabric Loader
3. Install Cobblemon
4. Install Radical Cobblemon Treinadores
5. Drop the **Owl's Nest / A Toca da Coruja** `.jar` file into your `/mods` folder

## Requirements

- Minecraft 1.21.1
- Fabric Loader
- Cobblemon
- Radical Cobblemon Treinadores
